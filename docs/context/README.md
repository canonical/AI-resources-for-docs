# Context engineering

This folder contains examples of context engineering for LLMs. If you want to add an example from your own repo, make sure to remove any confidential sections first.

Include the name of your project in the name of the example file to avoid conflicts, e.g. `copilot-instructions-ubuntu-server.md`
