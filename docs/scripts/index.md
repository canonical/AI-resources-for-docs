# Helper scripts

These scripts have been created by LLMs to help us to maintain or improve documentation, typically by automating routine tasks.

Note that the prompts provided here have not been battle-tested or refined so your mileage may vary.


## Update redirects

This script and associated GitHub action will take the output of the linkchecker and automatically update the redirecting URLs to their new target locations.

```{toctree}
:maxdepth: 1

update_redirecting_urls/README_redirect_script
```
