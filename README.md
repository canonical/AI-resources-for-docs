# AI Resources for Documentation

This repository acts as a pool for our ongoing AI experiments in documentation.
It holds various re-usable prompts, LLM-generated scripts and other tools to
facilitate our work on documentation.

## Learning resources

- https://learn.github.com/skills, *Take flight with GitHub Copilot* - an end-to-end collection of beginner-level practical exercises that leverage GitHub Copilot. Most approaches exercised there, however, are universally applicable.
